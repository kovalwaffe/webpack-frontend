# Webpack Frontend Starterkit

A smart webpack based frontend project.

### Installation

```
npm install
```

### Start Dev Server with HRM

```
npm start
```

### Start Dev Server 

```
npm watch
```

### Build Prod Version

```
npm run build
```

### Features:

- ES6 Support via [babel](https://babeljs.io/) (v7)
- SASS Support via [sass-loader](https://github.com/jtangelder/sass-loader)
